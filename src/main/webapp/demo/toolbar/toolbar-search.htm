工具条演示


演示如何创建带特殊按钮的工具条，可以使用简写OZ.TB代替OZ.Toolbar，API文档参考 OZ.Toolbar API


注：按Ctrl+Home键切换控制台的显示


(默认)通用带搜索工具条，搜索按钮在右侧

通用带搜索工具条，搜索按钮在右侧(带高级搜索和清空按钮)

搜索按钮按常规按钮定位的工具条

代码参考：

var tb01 = new OZ.TB({
	id: 'tbContainer01',
    buttons: ["|",
	    ...,
	    {type:'search',
	    	onSearch:function(value){
	    		
	    	}
	    }
    ]
});
var tb01 = new OZ.TB({
	id: 'tbContainer01',
    buttons: ["|",
	    ...,
	    {type:'search',
	    	location:"inner",
	    	value:"办公室",
	    	btns:"search,advance,clean",
	    	onSearch:function(value){
	    		
	    	},
	    	onAdvance:function(value){
	    		
	    	},
	    	onClean:function(value){
	    		
	    	}
	    }
    ]
});
